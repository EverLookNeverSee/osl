## Run Locally

Clone the project:

```bash
  git clone https://github.com/EverLookNeverSee/osl.git
```

Go to the project directory:

```bash
  cd osl/
```

Install the dependencies:

```bash
  pip3 install -r requirements.txt
```

Start the server

```bash
  streamlit run osl/run.py
```
